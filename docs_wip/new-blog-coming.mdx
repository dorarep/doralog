# 最初にブログを始めた理由
そもそものきっかけは姉からブログを作って欲しいと依頼されたのが始まりでした。

そのときに正社員を辞めフリーランスになったのもあり、自分も新しい収益軸を作ろうと思い作ったのがきっかけです。

# 技術的なお話
## Wordpressを選んだ理由

やはりこれだけ使われてるだけあって、Wordpressのエコシステムは圧倒的でした。

高クオリティのブログが一瞬で作れて、ブログに必要な機能もプラグインという形で着脱可能なのは素晴らしかったです。

特に画像の圧縮・リサイズや、アフィリエイトリンクといった機能が簡単に導入できるのは体験としてすごく楽ですね。

今まではサービスを完成させる時点にまで辿り着けずに辞めてしまうことが多々あったので、とにかくブログを作り公開するところだけにフォーカスするにあたりWordpressは頼りになりました。

## Wordpressの限界
できるだけ低コストで運営したかったのでGCPの低スペックサーバで動かしていたのですが、とてつもなく遅い。

ブログという静的なコンテンツにも関わらず、静的ホスティングや、FaaSと相性の悪いWordpressはどうしても構造的限界がありました。

Wordpress自体のコードがなかなか歴史を感じさせるもので、プラグインも綺麗に抽象化されてないので手を加えるのにはなかなか覚悟が必要です。
